# 🔍 Overview

This repository includes 50 Python challenges categorized into five levels:

  1  Beginner Level

  2  Intermediate Level

  3  Advanced Level

  4  Expert Level

  5  Bonus Challenges

Each challenge is implemented in a clean and beginner-friendly way. You'll also find individual README files for selected challenges explaining the logic and usage.

# 🧩 Challenge Categories
 ## 🟢 Beginner Level

    1. Number Guessing Game

    💡 Write a Python program that generates a random number between 1 and 100 and asks the user to guess it, providing feedback on whether     the guess is too high or too low.

    2. Palindrome Check

    💡 Write a Python function that checks if a given string is a palindrome (reads the same forwards and backwards).

    3. Simple Interest Calculation

    💡 Write a Python function that calculates simple interest given principal, rate, and time.

    4. Reverse a String

    💡 Write a Python function that reverses a given string without using built-in reverse functions.

    5. Find the Average

    💡 Write a Python function that calculates the average of a list of numbers.
  
 ## 🟡 Intermediate Level

    6. Matrix Addition

    💡 Write a Python function that adds two matrices.

    7. Character Frequency Count

    💡 Write a Python function that counts the frequency of each character in a string.

    8. String Reversal

    💡 Write a Python function that reverses the order of words in a string.

    9. Temperature Converter

    💡 Write a Python function to convert temperatures between Fahrenheit and Celsius.

    10. Duplicate Elements in List

    💡 Write a Python function that finds and returns duplicate elements in a list.
 ## 🔵 Advanced Level

    11. Anagram Checker

    💡 Write a Python function to check if two strings are anagrams of each other.

    12. Towers of Hanoi

    💡 Implement a solution to the Towers of Hanoi problem using recursion.

    13. Simple Password Validator

    💡 Write a Python function that validates a password based on specific criteria (e.g., length, digits, uppercase letters).

    14. Merge Sorted Lists

    💡 Write a Python function that merges two sorted lists into one sorted list.

    15. Find Common Elements

    💡 Write a Python function that finds common elements between two lists.
  
 ## 🔴 Expert Level

    16. Simulate a Queue

    💡 Implement a queue using two stacks.

    17. LRU Cache

    💡 Implement an LRU (Least Recently Used) cache using a dictionary and a doubly linked list.

    18. Binary Search Tree

    💡 Implement a binary search tree with insertion, deletion, and search functionalities.

    19. Sudoku Solver

    💡 Write a Python function that solves a Sudoku puzzle using backtracking.

    20. Word Frequency in Text

    💡 Write a Python program that calculates the frequency of each word in a text file.

# 🛠️ How to Use

    1 Clone this repository:

      git clone https://github.com/yourusername/python-practice-challenges.git
      cd python-practice-challenges

    2 Browse to the folder for your desired challenge.

    3 Read the README.md file in the folder (if available).

    4 Run the Python file:
      python challenge_name.py or python challenge_number.py
